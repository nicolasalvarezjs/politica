Políticas de Privacidad y Términos y Condiciones

Este documento describe cómo la aplicación maneja la información personal y establece las condiciones de uso del servicio. Al utilizar la aplicación, el usuario acepta estas políticas y términos.

Recolección de Información

  • Podemos recopilar datos personales que los usuarios proporcionen al registrarse o al usar la aplicación.
  • Utilizamos esta información para ofrecer y mejorar el servicio.
  • No compartimos información personal con terceros, salvo que sea necesario para prestar el servicio o por requerimiento legal.

Uso de la Información

  • Los datos se utilizan para personalizar la experiencia del usuario y para comunicarnos con él.
  • Los usuarios pueden solicitar la eliminación de sus datos mediante los canales de contacto disponibles en la aplicación.

Seguridad de los Datos

  • Implementamos medidas de seguridad para proteger la información personal.
  • Sin embargo, no podemos garantizar una seguridad absoluta y el usuario acepta este riesgo.

Términos y Condiciones de Uso

  • El usuario debe utilizar la aplicación de forma lícita y respetuosa.
  • No se permite el uso de la aplicación para fines ilegales o que puedan causar daño.
  • Nos reservamos el derecho de suspender o cancelar cuentas que incumplan estas condiciones.

Cambios en las Políticas

  • Podemos actualizar estas políticas y términos en cualquier momento.
  • Se notificarán los cambios relevantes a través de la aplicación o mediante otros medios.

Contacto

  • Para dudas o solicitudes relacionadas con la privacidad, el usuario puede contactarnos a través de la información provista en la aplicación.
